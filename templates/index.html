Encrypt file

This feature allow user to encrypt a file. The user can select file with the "File" input box. The file is encypted with a random symmetric key. This key is encrypted by RSA assymetric key, provided by the user in the "Public key" input box. This encypted symmetic key is then put in the output file header. Lastly the file is encrypted by AES-GCM block cipher. The data needed for AES-GCM decryption are also included in the file header. Output file is in base64 encoding and can be downloaded by clicking on the link "Download file".

Decrypt file

Encrypted files can also be decrypted. User can provide file with the "File" input box. Firstly the information is loaded from the file header to decrypt with AES-GCM. If the file is corrupted the process is aborted. Next, the symmetric key from the file header is decrypted by the private key, corresponding to public key used to encrypt symmetric key. This private key can be loaded by the "Private key" input box. The output file is in the form of plain text and can be downloaded with the "Download file" link.

Generate keys

This website also allows user to generate assymetric key pairs, used to encrypt and decrypt by clicking on the "Generate keys" button. Keys can be then downloaded by clicking on the "Download public key" and the "Download public key" buttons.

Application

The offline application is available, if the user wish not to send his key to the website. It can be used to encrypt, decrypt and generatin RSA key pairs.
